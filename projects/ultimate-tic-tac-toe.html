Module 01

Ultimate Strategy Board Logic

Instead of a single grid, this game utilizes nine mini-grids within one large grid. Where you play on a mini-board determines which board your opponent must play on next, raising the level of strategy required.

  • Dynamic mini-board highlight indications.
  • Real-time server validated turn rules.

System Architecture

Frontend View

A dynamic, lightweight client using vanilla JavaScript and CSS layout templates. Communicates with Spring endpoints asynchronously to handle lobby lists, play updates, and dashboard profiles.

Application Engine (Spring Boot)

Handles primary game loops, matches active player sockets, validates the nested moves against grid rules, and serves as the API gateway.

Persistent Storage (MySQL)

Hosts relational databases mapping user credentials, global ranking statistics, total historical matchup records, and configuration controls.

Cloud Host (Google Cloud Platform)

Hosts backend deployment containers on VM instances (Compute Engine / App Engine) paired with Cloud SQL for managed, persistent database management.
