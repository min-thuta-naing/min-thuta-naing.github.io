Koalytics

Biometric Usability Testing Platform

Senior Project

Koalytics is a remote usability testing platform that integrates computer vision models to track, analyze, and visualize biometric user feedback. It captures facial expressions, predicts emotion, and logs interaction patterns to provide UX researchers with quantitative, objective usability data.

Module 02

Real-Time Facial landmark Tracking

During session recording, the webcam stream is analyzed using MediaPipe FaceMesh to plot key landmarks such as eye movement (gaze vectors) and expressions (mouth posture, eyebrow positioning).

  • Zero-latency facial feature plotting via client-side libraries.
  • Simultaneous screen-sharing and audio capture.

Module 03

Emotion Modeling & Analytics Visualization

Backend workers use DeepFace to classify emotional expressions over the duration of the test. The system aligns emotional highs and lows directly with specific event timestamps, plotting them on an interactive timeline.

  • Detailed task success and frustration metrics.
  • Heatmaps of user focus areas relative to webpage layouts.

System Architecture

images/projects/koalytics-architecture.png

Frontend (React Client)

Coordinates simultaneous audio, camera stream, and screen sharing recording. Leverages MediaPipe client-side FaceMesh for basic real-time eye and landmark plotting to guide the user stream format.

Backend (Django REST Framework)

Provides JSON endpoints for session settings and stores processed emotion and timeline events. Integrates task validation, handles token-based session authentication, and packages results.

Biometric AI Engine (OpenCV & DeepFace)

An offline/async pipeline that reads frames from raw session video recordings. Maps the landmark changes via FaceMesh and uses the DeepFace classification model to score basic expressions.

Database (MySQL)

Maintains relational schemas for project settings, task criteria, session timestamps, user lists, and pre-calculated statistics mapped with respective session keys.
